# Decision-Trees

## Package Used
1. Matplotlib
2. Pandas
3. Pydotplus (Optional)
4. Numpy
5. Os
6. Sklearn
## Dataset
Dataset can be accessed from the following link:
https://www.kaggle.com/datasets/jeevanrh/drug200csv
## Execution
The execution of this model can be done on both Jupyter notebook and Google Colab.
